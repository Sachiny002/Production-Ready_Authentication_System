# Production-Ready_Authentication_System
#  Production Authentication System

##  Overview
A production-ready authentication system built using Flask and React with JWT-based security.

## Features
- User Registration & Login
- JWT Authentication (Access + Refresh Tokens)
- Protected APIs
- Password Hashing (Bcrypt)
- Environment Variables (.env)

##  Tech Stack
- Backend: Flask
- Frontend: React
- Database: SQLite
- Auth: JWT

##  Security Features
- Secure password hashing
- Token-based authentication
- Protected routes

##  Run

Backend:
pip install -r requirements.txt
python run.py

Frontend:
npm install
npm start
